A tool that can change the sound into
- Distorted<br>
- Tupai<br>

- 8D Audio<br>
- Reverse Audio<br>
</p>

## Clone this repo

```bash
> git clone https://github.com/shansekai/sound-effect
```

## Install:

```bash
> cd sound-effect
> bash install.sh
> mv sound-effect /sdcard
> cd /sdcard/sound-effect
```

### Run
```bash
> node index.js
```
